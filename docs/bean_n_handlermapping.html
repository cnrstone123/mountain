Open Declaration org.springframework.web.servlet.handler.AbstractHandlerMapping Abstract base class for org.springframework.web.servlet.HandlerMapping implementations. Supports ordering, a default handler, handler interceptors, including handler interceptors mapped by path patterns. Note: This base class does not support exposure of the PATH_WITHIN_HANDLER_MAPPING_ATTRIBUTE. Support for this attribute is up to concrete subclasses, typically based on request URL mappings.
